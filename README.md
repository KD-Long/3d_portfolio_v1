## Overview

Personal portfolio

- React
- Three.js

## How to run

1) clone
2) npm install
3) create .env file to inlude email.js env variables (service,template id etc)
4) npm run dev





.env file example:
VITE_EMAILJS_SERVICE_ID="service_......"
VITE_EMAILJS_TEMPLATE_ID="template_....."
VITE_EMAILJS_PUBLIC_KEY="_...."
VITE_EMAILJS_MY_EMAIL="....@gmail.com"
